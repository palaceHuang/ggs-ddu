# 备份服务器修改OS密码后续

关键词： 83 bu6-BG_p  

1. 从备份服务器上删除下原有的ssh登录记录信息
   cd /root/.ssh/   或者 cd ~/.ssh/
   rm -f authorized_keys
2. 分别在需要备份的主机上重新输入备份服务器口令
   ssh-copy-id root@192.168.50.83
   输入83的新密码
   测试：ssh 'root@192.168.50.83'
